Help

Overview:

This is a text editor which has a built in outlining tool.

Keyboard shortcuts:

"CTRL+RIGHTARROW" to create new outline or level

"CTRL+UPARROW" to move an element upwards in an outline

"CTRL+DOWNARROW" to move an element downwards in an outline

"CTRL+P" to print current page contents
